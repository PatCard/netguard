# Fase 1 — Tool Use básico

## Objetivo

Implementar el chat web con el agente IA y las herramientas básicas de escaneo de red.

## Funcionalidades

- Chat web con el agente
- Herramienta: escanear hosts activos en la red
- Herramienta: consultar puertos abiertos de un host
- Guardar historial de escaneos en MySQL

## Stack utilizado en esta fase

| Capa | Tecnología |
|------|-----------|
| Frontend | React + Tailwind |
| Backend | Laravel (API REST) |
| Agente IA | Anthropic SDK + Python |
| Escaneo de red | Nmap + Scapy (Python) |
| Base de datos | MySQL |
| Contenedores | Docker Compose |
